Temperature Sensor Subsystem

Table 2: Temperature Sensor Selection

Solutions Pros Cons
Option 1
TC74A4-3.3VCTTR
$1.15 /per unit
Datasheet
  • Low operating current (250microA)
  • +/- 3 degrees Celsius accuracy between 0-125 degrees Celsius
  • Digital output (8bit)
  • Simple I2C interface
  • Cheap
  • Very tiny SOT-23 package, difficult for hand soldering
Option 2
AT30TS74-XM8M-T
$0.87 /per unit
Datasheet
  • Low operating current of 85 microA
  • +/- 1 degree Celsius accuracy of temperatures between -20 to 100 degrees Celsius
  • Simple I2C interface and SMBUS
  • 9-12 bit configurable resolution
  • 8 MSOP, easier hand soldering
  • Cheap
  • Takes up more space than other options
Option 3
SHT30-DIS-B10KS
$3.45 /per unit
Datasheet
  • Uses I2C communication
  • Accuracy of +/- 0.1 degrees Celsius
  • Quick measurement cycle
  • Compact
  • Reliable
  • No amplification, uses filter
  • Also measures humidity
  • DFN package, difficult hand soldering
  • Expensive

Choice:

We are choosing option 2.

Rationale:

Initially option 1 seemed ideal since we would be working with that component in class, and it is a digital temperature sensor with I2C communication. Option 2 is very similar to option 1, but it has a bit larger package which should be easier to hand solder. Additionally, it has a lower operating current, more accurate, and is more of a configurable resolution.
